## 4. 作業ステップ

1. ヒアリング内容をもとに画面構成を確定
2. `src/App.tsx` にセクション骨組みを実装
3. Tailwind / CSS で配色・タイポグラフィ適用
4. 素材（料理写真等）を反映
5. 文言と表示崩れを調整
6. `npm run lint` 実行
7. 必要に応じて `npm run build` 実行

- lint エラーがない
